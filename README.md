# 📊 Customer Trends Analysis Project

## 📌 Project Overview
This project analyzes customer shopping behavior using data analytics techniques.  
The goal is to understand customer patterns, spending habits, and product preferences.

---

## 🎯 Objectives
- Analyze customer purchasing behavior  
- Identify trends and patterns  
- Segment customers  
- Generate business insights
